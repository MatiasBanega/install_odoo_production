Instalar Odoo en producción en pocos pasos

Este Script permite instalar odoo de forma aislada para usar en producción.

Modo de uso

Simplemente cree un usuario que se llame pronexo, luego agregue al usuario al grupo sudo de la siguiente manera: # usermod -aG sudo pronexo, logueese como usuario pronexo, descargue el Script, de permisos de ejecución (chmod +x odoo17.install.sh)y comience la instalación que durará entre 2 a 10 minutos dependiendo la velocidad de su conexión a Internet!
